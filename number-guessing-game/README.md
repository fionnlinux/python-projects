# Dad's Number Guessing Game

A simple number guessing game written in Python as a first project.

## What it does

Generates a random number between 1 and 100 and prompts the player to guess it. Gives higher or lower hints after each guess, tracks attempts, and rates performance at the end. Validates input so non-numeric entries don't crash the game.

## Running it

```bash
python3 dads-number-guessing-game.py
```

Requires Python 3.

## What I learned

- Basic Python syntax and structure
- User input and type conversion
- Conditionals and loops
- Exception handling with try/except
- Random number generation with the `random` module
- Functions and docstrings
